# jxcapp-updates

用于给“小兰进销存”提供应用内更新清单和 APK 下载源。

## 目录

- `update.json`
  应用更新清单，主应用读取这个文件判断是否有新版本。
- `releases/`
  存放 APK 文件。

## 推荐发布流程

1. 把新 APK 放到 `releases/`
2. 更新 `update.json` 里的版本号、下载地址、发布时间和更新说明
3. 提交并推送到 GitHub

## 当前说明

当前仓库已初始化更新清单模板，但还没有上传可直接下载的 APK。
只有当 `apkUrl` 指向一个真正可直接下载的 APK 地址时，主应用内更新按钮才会生效。
